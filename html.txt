1.
HTML 是用来描述网页的一种语言。
HTML 指的是超文本标记语言 (Hyper Text Markup Language)
HTML 不是一种编程语言，而是一种标记语言 (markup language)
标记语言是一套标记标签 (markup tag)
HTML 使用标记标签来描述网页

2.HTML 标签
HTML 标记标签通常被称为 HTML 标签 (HTML tag)。
HTML 标签是由尖括号包围的关键词，比如 <html>
HTML 标签通常是成对出现的，比如 <b> 和 </b>
标签对中的第一个标签是开始标签，第二个标签是结束标签
开始和结束标签也被称为开放标签和闭合标签

3.HTML 文档 = 网页
HTML 文档描述网页
HTML 文档包含 HTML 标签和纯文本
HTML 文档也被称为网页
Web 浏览器的作用是读取 HTML 文档，并以网页的形式显示出它们。
浏览器不会显示 HTML 标签，而是使用标签来解释页面的内容：

4.常用排版标签
title:网页标题标签
h1~h6:忘本标题标签
p:段落标签
<br/>:换行标签
<hr/>:水平线标签
格式化标签

    i,em,cite:斜体标签
    sup:上标标签
    sub:下标标签

特殊符号标签
    &nbsp;  空格
    &quot;  "
    &gt;    >
    &lt;    <
    &copy;  版权符
    &reg;   注册商标符

5.元素的分类
1)块元素
    独占1行
    常见的有
        <h1>~<h6>,<p>,<div>,<hr/>,<ol>,<ul>...
2)行内元素
    和其他行内元素公用1行
    常见的有
        <strong>,<em>,<a>,<span>,<img/>,<input>,<font>...

注意:
    块元素可以嵌套块元素和行内元素
    而行内元素只能嵌套行内元素.

6.列表标签
一般页面的标题栏都是列表标签.
1)有序列表
    <ol>
        <li>列表项<li/>
    </ol>
2)无序列表
    <ul>
      	<li>列表项<li/>
    </ul>
3)定义列表
    <dl>
       <dt>定义名称</dt>
       <dd>定义名称描述</dd>
    </dl>

7.字体标签
    <font>

8.图像标签
    <img/>
        常用属性:
            src:图像的路径.
            alt:图像不能正常显示时的文字信息,主要给搜索引擎使用
            width:图像的宽度
            height:图像的高度.
            title:鼠标移动到图像上显示的文本信息,主要给用户看.

9.超链接标签
    <a>
    通常用于在不同页面之间进行跳转,也可以在当前页面
    常用属性:
        href:连接指向的地址,可以是本地地址,也可以跳转到当前页面的其他位置(锚点连接)
        target:打开方式
            _self:当前窗口打开
            _blank:新窗口打开

10.表格标签
    <table>:表格
    <tr>:行
    <td>:单元格
    <caption>:表格的标题
    <th>:表头单元格(居中+加粗显示)

表格语义化标签
    <thead>表头,<tbody>表体,<tfoot>表脚

    常用属性:
        width:宽
        height:高
        border:边框
        cellspacing:边框之间的间距
        cellpadding:单元格内容和单元格边框之间的间距
        bgcolor:背景色
        align:水平对齐方式.

    单元格合并
        rowspan:行合并
        colspan:列合并

11.表单标签
表单:用于在客户端收集信息,然后将信息提交给服务器处理
    <from>:表单标签
        action:表单提交地址
        method:表单提交方式
            get:默认方式,将表单数据直接提交给action指向的URL
                表单数据直接显示在浏览器地址栏中,安全性低
            post:表单数据包含在表单主体中,然后将表单主题提交给action指向的URL
                只能看到一些脚本信息,安全性比较高
        name:表单名称
        target:打开方式
            _blank:新窗口打开
        enctype:编码方式
            当需要上传文件的时候,需要设置该属性,且值必须为multipart/from-data

12.表单对象
1)input
    用于收集用户的信息,通过type属性设置不同的对象.
        type:表单类型
            1.1 text(单行文本输入框)
                value:值 maxlength:最大字符长度    size:文本框大小
                placeholder: 提示信息   readonly:只读 required:必须输入
            1.2 password(密码输入框)
                其他属性和text相同
            1.3 radio(单选框)
                checked:默认被选中
            1.4 checkbox(复选框)
                checked:默认被选中
            1.5 button(普通按钮)
                一般和JS一起使用
            1.6 reset(重置按钮)
                清空表单数据,回到初始状态
            1.7 submit(提交按钮)
                表单数据提交给服务器
            1.8 file(文件上传域)

            1.9 image(图片域)
            1.10 hidden(隐藏域)


2)textarea
    多行文本域
        rows:行数 cols:列数
3)select
    下拉列表
    <select>
        <option>列表项</option>
    </select>

        select属性
            size:可见的列表项个数
            multiple:设置多选
        option属性
            selected:默认被选中


注意:
1) radio,checkbox,select必须要添加name属性(分组),目的是为了和其他对象进行区分.
2) 其他对象上的name属性是用来表示该对象的,当提交表单数据的时候,必须要提供该属性.
3) 当提交表单的时候,所有对象都需要提交name属性.如果该表单对象无法录入数据,这时候
    需要提供value属性,(用于表示该对象的值).
